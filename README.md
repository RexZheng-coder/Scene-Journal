# SceneJournal

SceneJournal is a Swift Playgrounds app for capturing and revisiting memorable live-event moments with photos, locations, notes, highlights, and exportable journal entries.

## Project Structure

- `SceneJournal.swiftpm/`: main Swift Playgrounds app
- `docs/screenshots/`: challenge screenshots and presentation assets
- `archive/backups/`: local zip backups, excluded from git
- `archive/legacy/`: older experiments and unrelated project folders, excluded from git

## Git Notes

This repository is organized around the `SceneJournal.swiftpm` app. Build artifacts, local backups, and old experiments are kept out of version control so the repo stays clean and submission-ready.
